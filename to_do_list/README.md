# Project Board

Приложение управления проектами с Kanban доской и локальным хранилищем.

## Установка и запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173` в браузере.

## Что это?

Простой и быстрый инструмент для организации проектов. Проекты перемещаются между тремя статусами: **Новое**, **В процессе**, **Готово**.

Данные сохраняются в браузере автоматически.

## Для продакшна

```bash
npm run build
npm run preview
```
